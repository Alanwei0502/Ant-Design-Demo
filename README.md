# Introduction

It's an Ant Design Demo.

## Project Tree
```
antd-demo
├─── public
├─── src
│   ├─── assets
│   ├─── components
│   ├─── containers
│   ├─── i18n
│   ├─── layouts
│   ├─── service
│   ├─── store
│   ├─── utils
│   ├─── App.tsx
│   └─── index.tsx
```

## Folder Config

+ **assets** 圖片/第三方樣式
+ **components** 共用元件，例如：按鈕，表格，表單，卡片
+ **containers** 容器元件，主要負責資料的獲取，業務相關的互動等內容
+ **i18n** 國際化多語系配置
+ **layouts** 頁面中可複用的佈局，例如：導覽列、側邊選單
+ **service** 放置與資料互動相關的api
+ **store** 全域性狀態管理
+ **utils** 共用函式、工具類
